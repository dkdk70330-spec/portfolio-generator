Netlify에 사이트 올리기

  1. 배포 ZIP 저장

    상단의 백업을 누르고 Netlify 배포 ZIP을 선택하세요.

  2. Netlify Drop 열기

    Netlify 계정으로 로그인한 뒤 아래 버튼으로 배포 화면을 여세요.

  3. Netlify 업로드

    저장된 Netlify 배포 ZIP 또는 압축을 푼 파일을 업로드 영역에 끌어놓으세요. 반드시 index.html가 바로 보이는 파일이어야 합니다.

  4. 사이트 주소 확인

    배포가 끝나면 발급된 netlify.app 주소에서 사이트를 확인하세요.

  5. 사이트 업데이트

    내용을 수정한 뒤 새로운 배포 ZIP을 만들고, 기존 Netlify 사이트의 배포 화면에 다시 올리세요.
